Turtle Game

  • Institution
    Upper Austria University of Applied Sciences
  • Year
    2021
  • Type
    Game
    Interactive Installation
  • Software
    Unity 3D
  • My Part
    Concept
    Prototyping
    Implementation

In this project, our aim was to create an interactive game with a unique user interface. Our research led us to discover a project that projected colorful layers on a sandbox based on its height. Inspired by this, we came up with the idea of creating a multiplayer game that utilized sand as a user interface to control characters.

In this game, players control turtles by shaping the sand around them. By creating holes, the turtles slide naturally into them and can be pushed further using the sand. The objective of the game is to collect stars hidden under the sand, which can also be stolen from other players. The first player to collect five stars wins the game.

We developed a physics system that complemented the sand interface, making it intuitive and user-friendly as some user tests proved.

I was involved in the planning process of the game mechanics and logics, as well as the creation of the user interface. Additionally, I conducted numerous prototypes and created the physics system for the movement of the turtles.
