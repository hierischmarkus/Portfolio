Mediawall

  • Institution
    17K GmbH
  • Year
    2018
  • Type
    Interactive Installation
  • Software
    VVVV
  • My Part
    Implementation

For the BAUMA 2018 in Shanghai and the BAUMA 2019 in Munich my company 17K implemented several interactive exhibits for the tunnel boring company Herrenknecht AG.

One of my key responsibilities during this period was the development of the interactive application known as "Mediawall." This dynamic application served as a comprehensive showcase of Herrenknecht AG's complete product portfolio. In China, it featured four Full HD touch screens, while in Munich, it boasted four 4K touch screens.

The Mediawall offered users the ability to explore detailed information about each product by simply clicking on it, which redirected them to the corresponding webpage on Herrenknecht AG's official website. To ensure an engaging experience, the application incorporated a screensaver that would activate when the user remained inactive for a certain period, inviting them to further explore the range of products.

Despite being an intern and working student, I was given a high responsibility for the implementation of the Mediawall, handling the majority of the implementation process. My role included the creation of a downloader tool responsible for retrieving all media data from a database, which was based on the Herrenknecht AG website. Additionally, I developed an application that filtered and organized the information from this database, presenting downloaded images alongside their related texts in a sorted grid layout. Furthermore, I successfully implemented the screen designs provided by a design agency.
